# 🌐 Network Traffic Anomaly Detection

## Overview
This project focuses on detecting anomalies in network traffic using machine learning techniques.

## Dataset
Synthetic network traffic dataset from Kaggle.

## Features
- Data preprocessing
- Feature engineering
- Anomaly detection

## Approach
- Cleaned and prepared dataset
- Applied machine learning model to identify abnormal patterns

## Output
- Detects unusual or suspicious network activity
