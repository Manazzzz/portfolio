# 📰 Fake News Detection using Machine Learning

## Overview
Built a machine learning model to classify news articles as real or fake using Natural Language Processing techniques.

## Dataset
- Combined dataset of real and fake news articles
- Text-based data (news content)

## Preprocessing
- Text cleaning (removing punctuation, stopwords)
- Tokenization
- Lowercasing

## Feature Engineering
- TF-IDF Vectorization

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting

## Evaluation
- Accuracy Score
- Model comparison across multiple algorithms

## Results
- Ensemble models (Random Forest, Gradient Boosting) performed best
- Achieved high accuracy on test data

- Use deep learning (LSTM, BERT)
- Deploy using Flask/Streamlit
